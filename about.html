Since then, numerous developers have contributed to its source code.

Website and release security

Developers do not have direct, independent write access to the website. All changes must be signed by both ThomasV and SomberNight, and are automatically checked by a script before being made publicly visible. Electrum release files are reproducible and must be signed by at least these two keys before the script makes them accessible to the web server.
